# ChristopherGonzales.github.io

Mechanical engineering portfolio website for resume sharing, project highlights, and internship applications.

Live portfolio themes now emphasize internship applications for automotive, aerospace, and industrial engineering roles.

## Customize your site

1. Open `index.html` and update contact links (email + LinkedIn).
2. Create an `images/` folder in the repository root.
3. Add your photos and name them:
	- `me.png` or `me.jpg` (profile photo in repo root)
	- Example gallery files currently wired: `Mustang.jpg`, `blueprintnasa.jpg`, `HeatSink.jpg`, `code.png`, `results.png`, `PantherCar.jpg`
4. Add your resume file as `ChristopherG.pdf` in the repository root so the Download Resume button works.
5. If your image names differ, update the `<img src="...">` paths in the `Engineering Gallery` section.
6. Push changes to GitHub.

## Local preview

Open `index.html` in your browser, or run a quick local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Publish on GitHub Pages

If this repository is named `ChristopherGonzales.github.io`, your site publishes automatically at:

`https://christophergonzales.github.io`

If needed, go to repository **Settings > Pages** and ensure deployment is set to the default branch.
